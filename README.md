


## GUIA DE USO

en el package.json:

"scripts": {
    "start": "node app.js",
    "dev": "node --watch app.js",
    ...
},

"type": "module"

no se olvide de instalar: 

- npm install express

- npm install pug

- npm install dotenv

- npm install --save sequelize

- npm install -- save pg pg-hstore
